# Zip of Secrets

**Category:** Forensics  
**Difficulty:** Easy  
**Points:** 100

## Description
A password-protected zip file hides a secret. The password might be easier to guess than you think.

## Provided Files
- `secret.zip` — An encrypted zip file containing flag.txt

## Objective
Crack the zip password and retrieve the flag.

## Setup
```bash
pip install pyzipper
python create.py
```
