# Steg-anography

**Category:** Forensics  
**Difficulty:** Medium  
**Points:** 250

## Description
A picture is worth a thousand words — but this one hides a few more. Something is buried in the pixels.

## Provided Files
- `stego.png` — A PNG image with a hidden message

## Objective
Extract the flag hidden in the image using LSB steganography.

## Setup
```bash
pip install -r requirements.txt
python embed.py
```
