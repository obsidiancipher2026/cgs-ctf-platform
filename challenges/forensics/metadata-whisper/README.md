# Metadata Whisper

**Category:** Forensics  
**Difficulty:** Easy  
**Points:** 75

## Description
Someone whispered a secret into this image's metadata. Can you find it?

## Provided Files
- `photo.png` — An innocent-looking PNG image

## Objective
Find the hidden flag in the image metadata.

## Setup
```bash
pip install -r requirements.txt
```
