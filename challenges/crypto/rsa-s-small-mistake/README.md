# RSA's Small Mistake

**Category:** Crypto  
**Difficulty:** Medium  
**Points:** 300  

## Description

Someone thought using small primes would be efficient. They were wrong.

## Files

- `rsa_params.txt` — RSA public parameters (n, e, c)

## Hints

- n is small enough to factor with the right tools.
- YAFU, msieve, or SageMath can handle numbers this size.
