Predictable Token Generator
===========================

A web application generates "random" session tokens using the following code:

```python
import random
import time

def generate_token():
    # Seed with current Unix timestamp
    seed = int(time.time())
    random.seed(seed)
    token = ''.join(random.choices('abcdef0123456789', k=32))
    return token
```
